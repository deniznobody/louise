# **Louise** - the next generation multipurpose discord bot.

## Features

- Database integrated LastFM system.
- ID based punishment system.
- Built-in Vanity Protector.
- Built-in Anti Raid System.
- Web Dashboard.
- Custom Commands.
- Welcoming Messages.
- Reaction Roles.
- XP System.
- Poll & Giveaway System.
- and much more.
